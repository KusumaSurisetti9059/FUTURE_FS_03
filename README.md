  Sri Durga Karthikeya Metal Stores Website

This project is a responsive business website developed for a real metal and home appliances store using HTML, CSS, and JavaScript.
The website was created as part of the Future Interns Frontend Development Internship Task.


Project Objective:
The objective of this project is to provide an online presence for the business and showcase products, business details, and contact information in a professional and user-friendly manner.



Features

- Responsive Design
- Modern User Interface
- Product Showcase Section
- Hover Effects
- Product Expansion Feature
- Detailed About Us Section
- Contact Section with Location
- Professional Layout



Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages



What I Learned

- Frontend Web Development
- Responsive Web Design
- JavaScript Interactivity
- GitHub Deployment
- Debugging and Problem Solving
